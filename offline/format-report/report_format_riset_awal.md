# 1. Tujuan Riset
- Mengetahui kebutuhan calon peserta.
- Menentukan tema kegiatan yang relevan.
- Menentukan target peserta kegiatan.
- Menentukan format pelaksanaan kegiatan.

# 2. Metode Riset
| Aspek            | Keterangan                            |
| ---------------- | ------------------------------------- |
| Metode Riset     | Survey                                |
| Media Riset      | Google Form                           |
| Target Responden | Mahasiswa / Pelajar / Umum            |
| Jumlah Responden |                                       |
